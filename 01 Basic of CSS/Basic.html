What is CSS

ANSWER:

  1. CSS stands for Casecading Style Sheet
  2. CSS is the language we use to style an HTML document.
  3. CSS describes how HTML elements should be displayed.
  4. Used for beautify HTML strcuture
  5. Ways to adopt/apply CSS in html tags

Now let try to break the acronym:

  1. Cascading: Falling of Styles
  2. Style: Adding designs/Styling our HTML tags
  3. Sheets: Writing our style in different documents

Why Use CSS?

CSS is used to define styles for your web pages, including the design, layout and variations in display for different devices and screen sizes.

Types of CSS

  1. Inline CSS (within tag with the help of style attribute) not recommanded as per SEO rules definade by [w3c], applied for single tag
  2. Internal CSS (within page with the help of style tag) not prefrabble by coding point of view applicable for single page
  3. External CSS most recommanded and preferabble needs to create .css exetension file for multiple pages we can apply extenal css data to HTML with <link rel="stylesheet" href="style.css">;
